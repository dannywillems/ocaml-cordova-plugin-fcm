Module Cordova_fcm.Data

module Data: sig .. end
The following module defines a type Cordova_fcm.Data.t which contains information about the notification (which is a JavaScript object). To access to properties of a value of type Cordova_fcm.Data.t, get functions are implemented.

In cordova-plugin-fcm, these data comes from the "data" payload in the JSON sent to FCM. See the official documentation: https://github.com/fechanique/cordova-plugin-fcm

type t = Ojs.t
The type representing the data object received with each notification
val was_tapped : t -> bool
was_tapped data returns true if the application was in background and the user taps the notification. If the application was in foreground, the value is false.
val from : t -> string option
The ID of the application. For FCM, it's the project number. NOTE: A value is set only if the application was in background. 
val collapse_key : t -> string option
Seems to be always the application ID. NOTE: A value is set only if the application was in background. 
val google_message_id : t -> string option
The topic message ID when GCM has successfully received the request and will attempt to deliver to all subscribed devices. NOTE: A value is set only if the application was in background. 
val get : t -> string -> Ojs.t
get data key returns the value of data.key as an Ojs.t type.
val get_opt : t -> string -> Ojs.t option
get_opt data key returns the value of data.key as an Ojs.t option. If key is not an attribute of data, it returns None.
